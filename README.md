<h1>Quick Start</h1>

```shell
git clone https://github.com/michaelhyi/michael-yi.com.git
cd michael-yi.com

yarn install
//or
npm i

yarn dev
```

<h1>Branches</h1>

- prod: production branch, deployed live via vercel; do not push commits or merge pr's
- dev: dev branch, do not push commits but merge pr's
